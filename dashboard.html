DASHBOARD

HTML, TailwindCSS, Laravel, Inertia, Vue.js, PHP, phpMyAdmin, XAMPP

Tijdens dit project heb ik me veel kunnen versterken in Laravel, Object-Oriented Programming en het werken met het Model-View-Controller-principe.


Ik heb verschillende modellen, views en controllers ontwikkeld om data efficiënt te beheren en te presenteren. Via Inertia en Vue heb ik interactieve componenten gebouwd die direct communiceren met de Laravel-backend. De database heb ik opgezet en beheerd met phpMyAdmin, waardoor ik meer inzicht kreeg in databasemodellering en relaties tussen tabellen.

Het project

Laravel-backend

Tijdens dit project lag mijn focus voornamelijk op de back-end ontwikkeling in Laravel. Ik heb me verdiept in het opzetten van models, controllers en routing, en in het efficiënt structureren van de applicatie volgens het MVC-principe. Dit hielp me om een overzichtelijke en onderhoudbare codebase te creëren.

Controllers

Een belangrijk onderdeel was de DashboardController, waarin ik data ophaal en verwerk voordat deze via Inertia naar de Vue-frontend wordt gestuurd. Door gebruik te maken van services, zoals de JiraService, heb ik geleerd om logica op een gestructureerde manier te scheiden van de controllers, wat de code beter schaalbaar en herbruikbaar maakt.

Authenticatie en API

Daarnaast heb ik gewerkt aan authenticatie, dataverbindingen en API-integraties, waardoor ik beter begrijp hoe verschillende lagen binnen een webapplicatie samenwerken. Ook heb ik de database ontworpen en beheerd via phpMyAdmin, wat me veel inzicht gaf in relaties tussen tabellen en datamodellering.

Scrum aanpak

Qua projectaanpak heb ik gewerkt met sprints, reviews en een scrumboard om mijn voortgang te plannen en bij te houden. Deze manier van werken hielp me om gestructureerd te blijven, prioriteiten te stellen en tussentijds feedback te verwerken. Hierdoor kon ik mijn code en workflow steeds verder verbeteren.
